# CHAPTER_11_DRAFT_SUMMARY

第11章「AI時代の判断力」を公開初稿相当へ拡張した。

## 中心メッセージ

```text
AIが仕事を速くするほど、人間には判断の質が問われる。
判断力とは、AIの出力を信じる能力でも、ただ疑う能力でもない。
AIが作った情報、選択肢、提案、反証を材料にしながら、
人間が目的、責任、価値判断、リスク、関係者、タイミングを引き受け、
採用、修正、棄却、保留、実験化、停止、やめる判断を下す能力である。
```

## 追加した主な内容

```text
・AI時代に判断力が必要になる理由

・AIに任せられる判断補助、任せられない判断
・採用、修正、棄却、保留、実験化、拡大、縮小、停止、やめる、方向転換、リスク受容の分類
・判断レベル J0〜J5
・Decision Brief
・判断前チェックリスト

・Option Evaluation Table

・Kill Decision Sheetによるやめる判断
・実験化判断
・Risk Acceptance Memo
・Decision Review Log
・判断ワークフロー
```
