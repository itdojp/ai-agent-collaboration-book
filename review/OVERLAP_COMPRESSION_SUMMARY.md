# 第15章・第16章 重複圧縮サマリー

版: `0.20.0-revised`  
対象: 第15章「スキルマップと評価」、第16章「組織展開と教材化」

## 反映内容

第15章と第16章の責務を以下のように分離した。

| 項目 | 第15章 | 第16章 |
|---|---|---|
| 主目的 | 能力を定義し、評価する | 評価結果をもとに展開・運用する |
| 主な成果物 | Skill Map、Role Skill Profile、Evidence Portfolio、Learning Gap Handoff、Certification Decision Record | Role-based Learning Path、Curriculum Blueprint、Office Hour Runbook、KPI Dashboard、90-Day Rollout Plan |
| 学習 | 不足領域と再評価成果物を定義する | 教材、ワークショップ、OJT、Office Hourとして提供する |
| 認定 | 合格基準と判定記録を定義する | 導入順序、権限付与、台帳反映、再評価通知を運用する |
| KPI | 評価に必要な証拠を定義する | 組織展開ダッシュボードとして運用する |
