# 用語・表記レビュー

## 1. 結論

本書の主要用語は概ね安定している。ただし、公開前に以下を統一すべきである。

1. レベル接頭辞の衝突を解消する。
2. Source Notes見出しを統一する。
3. 日本語名 / 英語名の初出ルールを明確にする。
4. 付録Eに全成果物名を登録する。
