# 付録D: 職種別ユースケース集

この付録は、AIエージェント協働の導入候補を職種別に整理するためのカタログである。各ユースケースは、単なるアイデアではなく、関連章、必要成果物、禁止事項、承認条件、リスクレベルまで含めて扱う。

リスクレベルは目安である。自社の情報分類、顧客契約、業界規制、社内規程がある場合は、それを優先する。

| リスク | 目安 | 必要な管理 |
|---|---|---|
| 低 | 社内下書き、個人メモ、公開情報整理 | Request Contract、自己レビュー |
| 中 | 社内データ、顧客文面案、チーム共有 | Context Pack、AI出力レビュー、ログ |
| 高 | 顧客影響、個人情報、契約、CRM更新、外部送信 | HITL、Security Checklist、AI台帳、Risk Register |
| 禁止 | 認証情報投入、無承認外部送信、自動契約判断、本番破壊操作 | 実施不可。例外は第10章の例外管理へ |
